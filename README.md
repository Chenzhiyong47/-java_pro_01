##### Windows 和 Linux 运行 Java 程序的异同
##### Linux java file 这一步没有后缀 ".java"， 而 Windows 是有后缀的， 多加了后缀，运行会报错
```
Windows:
    javac file.java; java file.java

Linux:
    javac file.java; java file
```
